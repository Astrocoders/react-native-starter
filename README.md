# react-native-starter


## Setup
```
yarn install
```

CocoaPods
```
cd ios/
pod install
```

If you don't have CocoaPods, install with RubyGem

```
gem install cocoapods
```

rnStarter App

## Running
```
yarn start
```
and
```
yarn bsb -make-world -w
```
